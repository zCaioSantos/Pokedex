# Contributing

Clone de repository, open its folder and install dependencies with:

```sh
npm i
```

Run it using:

```sh
npm start
```

## Dependencies

- Node.js
- Next.Js
- Axios

## Features

- [ ] Listar todos os Pokemons da 1ªGeração
- [ ] Detalhes dos Pokemons
- [ ] Lista de Pokemon capturados
