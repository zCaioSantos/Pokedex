# Contributing

Clone de repository, open its folder and install dependencies with:

```sh
npm i
```

Run it using:

```sh
npm start
```

## Dependencies

- Node.js
- Next.Js
- Axios
- React Icons

## Features

- [x] List all 3rd Generation Pokemon
- [x] Fetch Pokemon
- [x] Pokemon details
