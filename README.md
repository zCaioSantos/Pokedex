# Contributing

Clone de repository, open its folder and install dependencies with:

```sh
npm i
```

Run it using:

```sh
npm start
```

## Dependencies

- Node.js
- Next.Js
- Axios
- React Icons

## Features

- [x] Listar todos os Pokemons da 1ªGeração
- [x] Buscar Pokemon
- [x] Detalhes dos Pokemons
